Detailed Description

JSON functions, mostly used by pre-generated files.

Author
Amos Buchanan
Version
1.0
Date
2020

These are helper functions for parsing JSON. This is mostly useful for the auto-generated code by the preprocessor when JSON tags are used. This library depends on the JSMN JSON parser. There is a slightly modified version of the library included, which fixes an issue with including the header and source code separately. It is otherwise unchanged from the original.

This is a single-file library. You may include it as a header just as any other. Add the following define to include the source once per project:

#define JSON_SRC
#include "ab_json.h"

Flags for how to create a JSON string from a struct/enum.

These flags may be or'd together for input.

Enumerator
JSON_Null 

No flag.

JSON_IsLastInList 

The item is the last in a list; don't put a comma after it.

JSON_DontUseTag 

Don't use a starting tag to define this. Most often used for arrays.

JSON_BaseObject 

This is the root object of the JSON string.
