\subsection{Detailed Description}
A basic preprocessor to generate a header file. 

This is a basic pre-\/preprocessor that reads in the source files in a directory and outputs a generated source file with a number of standard functions. For usage, see the project \hyperlink{Readme_8md}{Readme.\+md} file.
