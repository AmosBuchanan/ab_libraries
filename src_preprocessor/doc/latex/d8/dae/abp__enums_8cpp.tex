This is the source file for the tags used in enum class. All enum class tag definitions go here. There are no tags for C-\/style enums; any C-\/style enums with a \mbox{\hyperlink{PreprocTest_8h_a2606cd56d2d8f567785bde5848176722}{T\+A\+G()}} will be ignored.

See the \mbox{\hyperlink{index}{Readme.md}} file for more information on usage.

To add a new tag\+:
\begin{DoxyItemize}
\item Write a function to create the header and function portion of the enums.
\item Update the \mbox{\hyperlink{abp__enums_8cpp_a9de7ea142237a33f158a301d823bfe35}{Process\+Enums}} function with the tag and new function.
